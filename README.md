# HexSoftwares_Titanic_EDA
Exploratory Data Analysis (EDA) on the Titanic dataset using Python
## Titanic Exploratory Data Analysis (EDA)

This project explores the Titanic dataset to understand factors that influenced passenger survival.
The analysis includes data cleaning, handling missing values, and visualizations to uncover insights.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Key Steps
- Data cleaning and preprocessing
- Exploratory data analysis
- Visualizations of survival patterns
